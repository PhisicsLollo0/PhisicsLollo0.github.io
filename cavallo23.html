Catalogue of Open Clusters (Cavallo+2023b)

Welcome to the repository of the extensive open cluster catalogue from the study Cavallo+23b. The complete dataset is available for download below, as featured in our recent publication.

In the table cavallo23.csv are listed the 6413 clusters with the parameters estimated by our ANNs. The README file contains the description of each column of the table cavallo23.csv
